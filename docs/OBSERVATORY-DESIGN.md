## Color Scheme

### Background Colors
- **Primary Background**: #0f172a (Very Dark Blue)
- **Card Background**: #1e293b (Dark Blue-Gray)
- **Card Gradient**: Linear from #1e293b to #334155
- **Border**: #334155 (Medium Blue-Gray)

### Text Colors
- **Primary Text**: #e2e8f0 (Light Gray)
- **Secondary Text**: #94a3b8 (Medium Gray)
- **Muted Text**: #64748b (Dark Gray)

### Status Colors
- **Success/Online**: #10b981 (Green)
- **Warning**: #f59e0b (Orange)
- **Error/Offline**: #ef4444 (Red)

### Accent Colors
- **Primary Blue**: #3b82f6
- **Secondary Purple**: #6366f1
- **Yellow (Cost)**: #fbbf24

### Chart Colors
- **Line 1**: #3b82f6 (Blue)
- **Line 2**: #10b981 (Green)
- **Bar**: #6366f1 (Purple)
- **Doughnut**: [#3b82f6, #10b981, #f59e0b, #8b5cf6]

## Interactive Elements

### Buttons
- **Primary**: Blue (#3b82f6) with hover darken to #2563eb
- **Secondary**: Purple (#6366f1) with hover darken to #4f46e5
- **Padding**: 0.5rem 1rem
- **Border Radius**: 0.375rem (rounded)

### Cards
- **Base**: Dark gradient background
- **Hover**: Translate up 2px, add shadow
- **Transition**: 0.2s ease-in-out

### Status Indicators
- **Size**: 8px circle
- **Animation**: Pulse (2s infinite)
- **Position**: Top-right of agent cards

### Tabs
- **Default**: Transparent with bottom border
- **Active**: Blue bottom border (2px)
- **Hover**: Darker background

## Typography

### Font Sizes
- **H1 (Page Title)**: 2.25rem (36px) - Bold
- **H2 (Section)**: 1.5rem (24px) - Semibold
- **H3 (Card Title)**: 1.25rem (20px) - Semibold
- **Body**: 1rem (16px) - Regular
- **Small**: 0.875rem (14px) - Regular

### Font Family
- **Primary**: System fonts (Tailwind default)
- `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, ...`

## Responsive Design

### Breakpoints
- **Mobile**: 1 column grid (<768px)
- **Tablet**: 2 column grid (768px-1024px)
- **Desktop**: 3 column grid (>1024px)

### Mobile Optimizations
- Stack charts vertically
- Collapse navigation to hamburger
- Reduce card padding
- Smaller font sizes

## Animations

### Page Load
- Cards fade in with stagger effect
- Charts animate on first render

### Real-time Updates
- Metrics fade and update smoothly
- Status indicators pulse
- New data slides in from bottom

### User Interactions
- Button press effect (scale down slightly)
- Card hover lift
- Chart hover tooltips

## Accessibility

- **ARIA Labels**: All interactive elements
- **Keyboard Navigation**: Tab through all controls
- **Color Contrast**: WCAG AA compliant
- **Screen Reader**: Descriptive labels

## Browser Support

- **Chrome/Edge**: 90+
- **Firefox**: 88+
- **Safari**: 14+
- **Mobile Safari**: 14+
- **Mobile Chrome**: 90+

## Performance

- **First Paint**: <1s
- **Interactive**: <2s
- **Chart Render**: <500ms
- **WebSocket Latency**: <100ms

---

This design creates a professional, modern monitoring dashboard that feels like enterprise tools (Grafana, Datadog, New Relic) while maintaining the BSM platform's aesthetic.
