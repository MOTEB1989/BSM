## üìã Communication Guidelines

### For Users

1. **Use GitHub Issues** for bugs and feature requests
2. **Use GitHub Discussions** for questions and ideas
3. **Join Telegram channels** (when available) for quick help
4. **Check documentation** before asking questions
5. **Be respectful** and follow code of conduct

### For Contributors

1. **Follow PR guidelines** in [GOVERNANCE.md](../GOVERNANCE.md)
2. **Run validation** before submitting (`npm run pr-check`)
3. **Write clear commit messages**
4. **Update documentation** for your changes
5. **Respond to review feedback** promptly

### For Administrators

1. **Use ORBIT bot** for system management
2. **Monitor security alerts**
3. **Review PRs** according to governance rules
4. **Moderate community channels**
5. **Maintain documentation**
